This is gonna be a tool that shows you some useful stuff for persona 5 royal. I think it would be nice to have
a day to day run down of what's possible for each day, and there's lots of awesome guides out there already for that,
but I think this one will aggregate a lot of different info and show it in a format that's easier to look up what day you're
actually at in the game.

Things I want to include:
-jazz club skill days
-weather
-major plot events (toggleable for spoilers probably)
-new sudoku puzzle days

possible adds:
-social link days
-whether a day has afternoon/night free time

My goal for this project is to create something that I would use, and is actually helpful, but also teaches me
about react, azure, and broadens my knowledge of web development in general.

It was started as a ASP.NET + React template project in visual studio 2026, so you can see some server side code
as well as some leftover template code, but ultimately I don't see this program as needing a backend because it's just
going to serve static information. I will probably try to host it on as an azure hobby static app if it's done, not 
planning on monetizing it or anything.
